# 利用可见光传输信息的系统
厦门大学计算机网络课程项目

## 一、项目简介
本课程项目主要由 **发送端（编码器）** 与 **接收端（解码器）** 两部分构成，整体基于可见光传输实现文件通信。
总体流程：用户提供待传输的二进制文件 → 发送端将二进制数据编码为一系列图像并生成视频 → 在显示器上播放视频 → 使用手机拍摄回放视频 → 将拍摄得到的视频传回计算机 → 接收端对视频逐帧解码为图像并还原出原始二进制文件。

## 二、实现思路
核心思路为：将二进制文件编码为可视文字序列，接收端通过 OCR 识别文字内容，再逆向解码还原数据。

`example` 文件夹中的示例采用 PaddleOCR 本地部署模型进行验证：
- 使用大写字母 + 数字编码时，识别准确率可达 100%；
- 使用常用汉字编码时，识别准确率可达 98%。

经后续优化后，系统预计可实现 100KB/s 以上的传输带宽，并保持较高的传输可靠性与质量。
