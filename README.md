# 利用可见光传输信息的系统
厦门大学计算机网络课程项目
##一、项目简介
本课程项目由发送端（编码器）与接收端（解码器）两部分组成。总体流程：用户提供要传输的二进制文件 → 发送端将二进制编码为一系列图像并生成视频 → 在显示器上播放视频 → 用手机拍摄回放视频 → 将拍摄视频拷回电脑 → 接收端对视频解码为图像并还原二进制文件。
##二、实现思路
将二进制文件编码为文字序列，接收端使用OCR进行识别并解码。
example文件夹中的样例试用了PaddleOCR的本地部署模型，使用英文与数字编码时识别准确率达到100%，使用常见汉字编码时识别准确率达到98%。
在优化后预计可以实现100KB/s以上的传输带宽，同时保证较高的传输质量。
